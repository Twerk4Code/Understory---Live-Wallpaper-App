# Understory---Live-Wallpaper-App

🌲 Understory
Cinematic live wallpapers for macOS.

Understory transforms your desktop into a living environment. Built using Swift and Metal, it provides high-fidelity, immersive motion backgrounds designed to blend seamlessly into your workflow without draining your battery.

✨ Features
Metal-Accelerated: Optimized for Apple Silicon and Intel Macs to ensure smooth playback with low CPU usage.

Minimalist Interface: Stays out of your way so you can focus on your work.

High-Fidelity Visuals: Supports cinematic loops and generative scenes.

Native macOS Integration: Feels like a part of the OS, not an afterthought.

🚀 Getting Started
Prerequisites

macOS 14.0+ (Sonoma or later recommended)

Xcode 15+ (if you are building from source)

Installation

Go to the Releases page.

Download the latest Understory.zip.

Unzip the file and drag Understory.app into your Applications folder.

Right-click and select Open (you may need to allow the app in System Settings > Privacy & Security).
